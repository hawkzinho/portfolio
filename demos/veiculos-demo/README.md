# AutoGest Demo

Versão demonstrativa do sistema de gestão de veículos.

## Como funciona
- Mesma interface do projeto original
- Sem banco de dados
- Dados salvos no `localStorage` do navegador
- Ao recarregar a página, a demo continua com as alterações salvas no navegador atual

## Resetar os dados da demo
No console do navegador:

```js
resetDemoData()
```

## Publicação
Pode ser publicado direto no GitHub Pages ou Cloudflare Pages, sem backend.
